# Data Rules

- Default comparison is EV Basket, not S&P 500
- EV Basket = equal-weight average of EV-first stocks in the selected timeframe
- Big Auto = equal-weight average of Ford, GM, and Toyota in the selected timeframe
- All performance charts must use normalized returns, not raw stock prices
- Timeframes:
  - 1D
  - 1W
  - 1M
  - 90D
  - YTD
- Relative performance = stock return minus benchmark return over the same timeframe
- V1 should prefer a small, reliable symbol set over a broader but fragile one
- Use mock data first, then wire live data second
