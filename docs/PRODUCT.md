# EV Pulse - Product Brief

## Core idea
Build a clean, visually premium dashboard that tracks a curated set of EV-first stocks and compares them against an EV Basket Index, with an optional Big Auto comparison, across 1D, 1W, 1M, 90D, and YTD timeframes.

## Problem
People interested in EV stocks can see prices anywhere, but they cannot quickly understand which EV names are leading, which are lagging, and how EV-first companies compare with major automakers in one clean view.

## Core user question
Which EV stocks are strongest right now, and are EV-first names beating or lagging major auto brands?

## V1 stock groups

### EV-first
- Tesla
- Rivian
- Lucid
- NIO
- XPeng
- Li Auto

### Big Auto comparison
- Ford
- GM
- Toyota

## V1 features
- Timeframe toggle: 1D, 1W, 1M, 90D, YTD
- Normalized multi-line performance chart
- Comparison toggle: EV Basket, Big Auto, None
- Ranked performance table
- Clickable stock detail panel

## Non-goals
- No auth
- No portfolios
- No alerts
- No news feed
- No AI summaries
- No advanced stock screener

## Design direction
- Dark mode first
- Premium and minimal
- Spacious layout
- Strong typography
- Smooth micro-interactions
- No cluttered finance-terminal look
